# 🤖 AI Resume Screening & Skill Matching System

An AI-powered Applicant Tracking System (ATS) that analyzes resumes against job descriptions using Google Gemini AI. The application calculates an ATS score, identifies matched and missing skills, generates career recommendations, and provides downloadable reports through an interactive Streamlit dashboard.

---

## ✨ Features

- 📄 Resume Parsing (PDF & DOCX)
- 📋 Job Description Parsing
- 🤖 Google Gemini AI Integration
- 🎯 ATS Score Calculation
- ✅ Skill Matching
- 📊 Interactive Dashboard
- 📈 Plotly Charts
- 💼 Hiring Recommendation
- 🤖 AI Career Coach
- 📄 PDF Report Export
- 📥 JSON Report Export
